Twitterati Android application displays search results of all tweets related to HiPages.
It uses GSON library to parse the json search results from Twitter API for better performance.
The results are displayed with a custom Listview adaptor with the user profile image and tweet text
To run the application, use the apk and follow the android installation process to view it. 
The application was created with Android Studio and follows native standards. Use of external libraries is limited to keep it simple.
